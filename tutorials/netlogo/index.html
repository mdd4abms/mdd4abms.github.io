Preparing NetLogo to execute the simulations

NetLogo version 6.0 or later must be used. First, a folder called osmnlogo must be created on the path /Netlogo/app/extensions/, download the osmnlogo jar file(from the site) and save it on the folder created. Restart NetLogo if it is open. Now NetLogo will be capable of executing the simulations exported from the ABStractme Tool.
